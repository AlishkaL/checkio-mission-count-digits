Прочитай про рядковий метод str.isdigit(). Він повертає True якщо всі символи в рядку - цифри, інакше False: 

text = "1a2"
print(text.isdigit())

# output
False
Добре, але ж мені потрібно перевірити кожен символ в рядку. Як це зробити?
Використай вбудовану функцію map() з методом isdigit() і вхідним рядком як її аргументами: 

dat = map(str.isdigit, text)
Ця операція створює об'єкт map. Я не знаю як продовжити.
Якщо ти конвертуєш цей об'єкт map в список з list() конструктором, ти отримаєш: 

print(list(dat))

# output
[True, False, True]
Оскільки True - для символів-цифр, то як мені їх тепер порахувати?
Python рахує логічні значення True/False як 1/0. Тож ти можеш використати звичайну функцію sum(). І ця функція може працювати з об'єктом map як своїм аргументом! Немає потреби в перетворенні у список: 

sum(map(str.isdigit, text))
То як має виглядати моє остаточне рішення?
Це вже майже все. Лише додай визначення функції і return: 

def count_digits(text: str) -> int:

    return sum(map(str.isdigit, text))

Використай модуль регулярних виразів щоб знайти всі цифри з відповідною маскою і поверни довжину цього списку знахідок: 

import re


def count_digits(text):

    return len(re.findall(r"\d", text))
# або
    return len(re.findall('[0-9]',text))
Чітке покрокове рішення: ітерування по рядку, перевірка кожного символа і збільшення лічильника у випадку якщо символ - цифра: 

def count_digits(text: str) -> int:

    a = 0
    for i in text:
        if i.isdigit():
            a += 1
            
    return a
