Lesen Sie über die String-Methode str.isdigit(). Sie gibt True zurück, wenn alle Zeichen in string Ziffern sind, sonst False: 

text = "1a2"
print(text.isdigit())

Ausgabe:

False

Ok, aber ich muss jedes Zeichen in der Eingabezeichenkette überprüfen. Wie kann ich das tun?

Verwenden Sie die eingebaute Funktion map() mit der Methode isdigit() und dem Eingabe-String als Argument: 

dat = map(str.isdigit, text)

Dieser Vorgang erstellt das Objekt map. Ich weiß nicht, wie ich fortfahren soll.

Wenn Sie dieses Map-Objekt mit list() onstructor in eine Liste umwandeln, erhalten Sie: 

print(list(dat))

Ausgabe:

[True, False, True]

Da True für Ziffern steht, wie kann man sie jetzt zählen?

Python zählt boolesche Wahr/Falsch als 1/0. Sie können also die einfache Funktion sum() verwenden. Und diese Funktion kann mit einem Map-Objekt als Argument arbeiten! Sie müssen sie nicht in eine Liste umwandeln: 

sum(map(str.isdigit, text))

Wie sollte also meine endgültige Lösung aussehen?

Es ist fast alles. Fügen Sie einfach eine Funktionsdeklaration und eine Return-Anweisung hinzu: 

def count_digits(text: str) -> int:

    return sum(map(str.isdigit, text))

Verwenden Sie das Modul für reguläre Ausdrücke, um alle Zeichen mit Ziffernmasken zu finden und geben Sie die Länge dieser Liste von Fundstellen zurück. 

import re


def count_digits(text: string) -> int:

    return len(re.findall(r"\d", text))
# or
    return len(re.findall('[0-9]',text))

Klare Schritt-für-Schritt-Lösung: Iterieren Sie über die Zeichenfolge, prüfen Sie jedes Zeichen und erhöhen Sie den Zähler, falls das Zeichen eine Ziffer ist: 

def count_digits(text: str) -> int:

    a = 0
    for i in text:
        if i.isdigit():
            a += 1

    return a
