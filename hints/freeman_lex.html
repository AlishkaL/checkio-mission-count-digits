Since True is for digit characters, how to count them now?

Python counts boolean True/False as 1/0. So you may use simple sum() function. And this function can work with map object as its argument! No need to convert into list: 

sum(map(str.isdigit, text))

So, how my final solution should look like?

It is almost all. Just add function declaration and return statement: 

def count_digits(text: str) -> int:

    return sum(map(str.isdigit, text))

Use regular expressions module to find all character with digit masks and return length of this list of findings. 

import re


def count_digits(text: string) -> int:

    return len(re.findall(r"\d", text))
# or
    return len(re.findall('[0-9]',text))

Clear step-by-step solution: iterate over the string, check each character and increase counter in case character is digit: 

def count_digits(text: str) -> int:

    a = 0
    for i in text:
        if i.isdigit():
            a += 1

    return a
